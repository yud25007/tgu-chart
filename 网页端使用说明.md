# 思政学分记录表网页端自动生成器

## 启动

首次使用先复制 `.env.example` 为 `.env`，并填写访问密码：

```bash
cp .env.example .env
```

双击 `start_web.command`，或在终端进入本文件夹后运行：

```bash
AUTH_PASSWORD=your-login-password python3 app.py
```

启动后浏览器会打开本地网页，默认地址从 `http://127.0.0.1:8765` 开始自动寻找可用端口。

## 推荐流程

1. 填写活动名称、日期、加分类型。
2. 填写默认加分数量，例如 `0.2`。
3. 上传 Excel 名单，点击“导入 Excel”。
4. 在网页表格里检查识别结果。
5. 如果提示某一行缺少学号，直接补齐或删除该行。
6. 点击“生成 Word”。

如果 Excel 文件名类似 `6.22毕业典礼摆放椅子活动统计表.xlsx`，网页会自动推断日期和活动名称。

## Excel 格式

目前支持第一行作为表头的 `.xlsx` 名单，例如：

```text
班级    姓名    学号
电气2402    张三    2024000001
电气2503    李四    2025000002
```

也支持带“加分”“学分”“所获学分”等列；如果 Excel 里没有加分数量，就使用网页上填写的默认加分数量。

如果不想上传 Excel，也可以把名单粘贴到“批量粘贴”框，再点击“识别名单”。

导入和生成时会检查缺失学号、缺失班级和重复学号。

点击“生成 Word”后，程序会复制 `模板.docx` 并写入新文件，不会修改原模板。

生成文件会自动下载，同时保存一份到：

```text
output/doc/
```

同时会把活动信息和名单明细保存到：

```text
data/auto_form.sqlite3
```

网页底部“历史记录”可以载入旧活动继续编辑，也可以重新下载已生成的 Word。

## 填写位置

- 活动名称、日期：写入模板第一行。
- 加分类型：写入第二行最后一格，默认是“思政学分”。
- 姓名、班级、学号、加分数量：从第三行开始逐行写入。
- 明细超过模板已有行数时，会自动在“学院意见”上方插入新行。
- 多页时表格会重复活动信息和表头，方便后续审核。
